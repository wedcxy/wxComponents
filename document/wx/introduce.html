介绍

wxComponents是一套个人封装的微信小程序组件库，并不是通过NPM下载安装使用，而是正常的导入使用

使用步骤

1.点击下载组件库包 点击下载安装包 (opens new window) 🎁

2.解压压缩包，把 components 📒文件夹放入项目目录中，和 pages 📒文件夹平级

点击查看文件摆放位置

3.使用 components 组件，需要在每一个页面的 .json 配置的文件的 usingComponents 📌属性里面添加

// 这里使用的是D-hint组件
{
  "usingComponents": {
    "D-hint":"/components/D-hint/D-hint"
  }
}

4.在 .wxml 使用 D-hint 📔组件

<D-hint></D-hint>
